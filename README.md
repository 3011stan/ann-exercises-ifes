# Atividades de Redes Neurais

Repositório com atividades da disciplina de Redes Neurais do Mestrado em Computação Aplicada (PPComp), Instituto Federal do Espírito Santo (IFES).

## Notebooks

- `01_backpropagation_mlp.ipynb`: implementação de um MLP com backpropagation em Python/NumPy, com comparação de desempenho usando `MLPClassifier` do Scikit-Learn.
- `02_mnist_cnn.ipynb`: classificação do MNIST original com rede neural convolucional, baixando os arquivos IDX e realizando o pré-processamento manualmente.
- `03_forecasting.ipynb`: previsão de série temporal com a base Airline Passengers, usando janelas temporais, baseline ingênuo e uma rede densa simples.
- `04_imdb_transformers.ipynb`: análise de sentimento na base IMDB oficial da Stanford usando `pipeline` da biblioteca Hugging Face Transformers.

Os notebooks foram desenvolvidos para fins acadêmicos, com explicações em português e execução compatível com ambientes como Jupyter e Google Colab.
